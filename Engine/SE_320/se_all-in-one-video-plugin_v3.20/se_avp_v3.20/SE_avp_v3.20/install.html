PRE-INSTALLATION INSTRUCTIONS (TO-DO ON YOUR COMPUTER)

  1. Download JW FLV Media Player from http://www.longtailvideo.com/players/jw-flv-player/. Type your email and select "I agree to the non commercial license" checkbox. Unselect "Include the JW Viral Plugin" checkbox and press "Download JW FLV Media Player" button.

    Notice: You can not use JW FLV Media Player on a commercial site if you do not have Commercial License. You can order Commercial License from https://www.longtailvideo.com/players/order. If you are not sure whether your website is a commercial site or not, please visit order page and see the examples. Also, the All-in-one Video plugin contains a standard skin. If you want to customize your player's skins, you can order new ones from https://www.longtailvideo.com/players/order.

  2. Extract player.swf and yt.swf you just downloaded from http://www.longtailvideo.com/players/jw-flv-player/ and insert them on All-in-one Video Plugin "/files/" directory. If you have download any skins, extract the file with extension *.swf (ex: modieus.swf) on "/files/include/vid_skins/" directory.

INSTALLATION INSTRUCTIONS - PLEASE FOLLOW THIS GUIDE CAREFULLY!

  1. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server, EXCEPT for install.html and license.html. Be sure to upload the files to the directory on your server where you have already installed SocialEngine. Upload all files in the "/images/" directory in BINARY mode - player.swf and yt.swf must also be uploaded in BINARY mode. Upload all other files in ASCII mode.

  2. Set full permissions (CHMOD 777) on the "/uploads_vid/" directory and on all subdirectories inside "/uploads_vid/".

  3. Login to your SocialEngine admin panel. On the "View Plugins" page you will see the plugin. Click the install link. Your installation is now complete!

  4. Be sure to delete the file "admin/install_vid.php" when you are finished. Leaving this file intact poses a serious security risk!

GETTING STARTED

  1. Modify the FFmpeg and FLVTool2 configuration in the admin panel. (Global Video Settings)
  2. Edit the video user level settings.
  3. Make sure that the maximum upload filesize is the same in your php.ini file.
  4. Also, make sure that your php.ini is configured correctly to handle big file uploads.
  5. Before uploading videos make sure that your SE directory is NOT password protected.

3.X -> 3.20 UPGRADE INSTRUCTIONS

  1. Make a COMPLETE BACKUP before you begin.

  2. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server by replacing the old ones, EXCEPT for install.html and license.html. Be sure to upload the files to the directory on your server where you have already installed SocialEngine. Upload all files in the "/images/" directory in BINARY mode - player.swf and yt.swf must also be uploaded in BINARY mode. Upload all other files in ASCII mode.

  3. Set full permissions (CHMOD 777) on the "/uploads_vid/" directory and on all subdirectories inside "/uploads_vid/".

  4. Please remove the files encode.php and config.php which have been replaced with new ones for security reasons.

  5. Login to your SocialEngine admin panel. On the "View Plugins" page you will see the plugin. Click the upgrade link. Your installation is now complete!

  6. Be sure to delete the file "admin/install_vid.php" when you are finished. Leaving this file intact poses a serious security risk! Also ensure that the All-in-one Video Plugin settings are still like you want them to be and your SE directory is NOT password protected.
