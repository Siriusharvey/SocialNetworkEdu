INSTALLATION INSTRUCTIONS - PLEASE FOLLOW THIS GUIDE CAREFULLY!

  1. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server, EXCEPT for install.html and license.html. Be sure to upload the files to the directory on your server where you have already installed SocialEngine.

  2. Set full permissions (CHMOD 777) on all the files in the "/templates" directory.

  3. With this version of the Ajax Photo Album, we have removed some of the old files that are not necessary anymore.
    You should delete the following ones:
    album_ajax_image.php (file)
    show_thumb.php (file)
    /thumbs/ (folder)

  4. Your installation is now complete!
