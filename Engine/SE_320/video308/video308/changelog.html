SocialEngine Video Plugin Version Changes

This file contains a list of each file that was modified in each new version as well as any language variables that were added or changed.

Version 3.08

  1. admin/install_video.php - Incremented version
  2. include/flow_player/flowplayer-3.1.0.swf - Removed
  3. include/flow_player/flowplayer-3.1.5.swf - Updated Flow Player to 3.1.5 (Upload in binary mode)
  4. include/flow_player/flowplayer.controls-3.1.0.swf - Removed
  5. include/flow_player/flowplayer.controls-3.1.5.swf - Updated Flow Player to 3.1.5 (Upload in binary mode)
  6. templates/styles_video.css - Changed pt font definition to px to ensure Windows DPI does not affect layout
  7. templates/video.tpl - Updated Flow Player to 3.1.5


Version 3.07

  1. include/class_video.php - Fixed thumbnail creation bug (improper arguments passed to -pix_fmt caused some ffmpeg versions to not properly create thumbnails)


Version 3.06

  1. admin/install_video.php - Incremented version
  2. templates/video.tpl - Fixed bug preventing playback in IE


Version 3.05

  1. admin/install_video.php - Updated the mod rewrite rules
  2. include/flow_player/* - Updated Flow Player to 3.1.0 (Upload in binary mode)
  3. templates/video.tpl - Updated Flow Player to 3.1.0; fixed reporting bug


Version 3.04

  1. header_video.php - Fixed bugs in permission checking, this will improve behavior on sites where video uploading is disabled
  2. user_video.php - Fixed permissions bug for sites with uploading disabled; small security fix
  3. user_video_upload.php - Fixed permissions bug for sites with uploading disabled
  4. video.php - Fixed permissions bug for sites with uploading disabled
  5. admin/install_video.php - Updated the mod rewrite rules and ffmpeg path checking code; fixed bugs in column creation when upgrading
  6. include/class_video.php - Tweaked video folder creation and added error silencing to prevent error messages in admin panel; fixed activity feed media
  7. templates/profile_video_tab.tpl - Fixed permissions bug for sites with uploading disabled; fixed profile display of YouTube videos when uploading is disabled
  8. templates/user_video.tpl - Fixed bug with ffmpeg path and showing upload link; made it possible to delete failed uploads
  9. templates/user_video_upload.tpl - Fixed permissions bug for sites with uploading disabled


Version 3.03

  1. header_video.php - Added code to utilize hooked template includes; changed to match new permission settings
  2. user_video.php - Added link to YouTube uploader
  3. user_video_upload.php - Fixed problem where sessions started by third-party plugins would conflict with and prevent upload (for <=3.07 only); added support for YouTube videos
  4. video.php - Added support for YouTube videos
  5. admin/admin_levels_videosettings.php - Added separate permission for YouTube videos
  6. admin/install_video.php - Incremented version
  7. images/icons/youtube.gif - Added icon
  8. include/class_video.php - Added YouTube support to video edit creation methods
  9. include/functions_video.php - Added function to extract a unique video ID from a YouTube URL
  10. templates/admin_install_video.tpl - Made it possible to install without FFmpeg after acknowledging a warning
  11. templates/admin_level_videosettings.tpl - Added separate option for YouTube videos
  12. templates/header_video.tpl - No longer included (provided for backwards compatibility)
  13. templates/user_video.tpl - Added link to YouTube uploader
  14. templates/user_video_upload.tpl - Added support for YouTube videos.
  15. templates/video.tpl - Added support for YouTube videos


Version 3.02

  1. browse_videos.php - Fixed bug that would sometimes cause query to fail for anonymous users; fixed support for "Public Permission Defaults"
  2. header_video.php - Fixed support for "Public Permission Defaults"; added site statistics hook
  3. user_video_upload.php - Session compatibility fix for SE 3.10; fixed bug with video search when disabled in admin panel
  4. admin/admin_header_video.php - Added site statistics hook
  5. admin/admin_levels_videosettings.php - Fixed bug in privacy handling that would update users in other levels; fixed checking no privacy options, will now replace it with only the most public
  6. admin/install_video.php - Incremented version; added a utility that runs "find / -name 'ffmpeg'" for convenience; converted action types, notify types, and system emails to use static language variable IDs for new installs; fixed language variable that would cause video thumbnail to not show up in activity feed
  7. include/functions_video.php - Added site statistics generation function; fixed bug that would cause all video action items to be private; action media path and link will now use relative URL
  8. templates/admin_install_video.tpl - Added a utility that runs "find / -name 'ffmpeg'" for convenience
  9. templates/admin_levels_videosettings.tpl - Made plugin loop in menu fully compatible with 3.10 (is backwards compatible with all v3 as well)
  10. templates/user_video.tpl - Fixed javascript error occuring if videos are forced to show up in search results


Version 3.01

  1. user_video_upload.php - Added support for the global flash uploader
  2. admin/admin_video_utilities.php - Added
  3. admin/install_video.php - Incremented version; fixed problem with code that checks if ffmpeg is installed; added support for retrying failed uploads; checks collation
  4. include/class_video.php - Added support for the global flash uploader; fixed some problems with wmv and mpg encoding; added debug code (disabled by default)
  5. include/functions_video.php - Added support for the global flash uploader; fixed some problems with wmv and mpg encoding; added debug code (disabled by default)
  6. templates/admin_video_utilities.tpl - Added
  7. templates/user_video.tpl - Added support for retrying failed uploads
  8. templates/user_video_upload.tpl - Added support for the global flash uploader with client-side file extension and size checking support
