INSTALLATION INSTRUCTIONS

  1. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server, EXCEPT for install.html, license.html and video_server_test.php. Be sure to upload the files to the directory on your server where you have already installed SocialEngine. Upload all files in the "/images/" and "/include/flow_player" directories in BINARY mode. Upload all other files in ASCII mode.

  2. Set full permissions (CHMOD 777) on the "/uploads_video" directory and all subdirectories contained in "/uploads_video".

  3. Login to your SocialEngine admin panel. On the "View Plugins" page you will see the plugin you have installed. Click the update link. Your upgrade is now complete!

  4. Be sure to delete the file "admin/install_video.php" when you are finished. Leaving this file intact poses a serious security risk!

GETTING STARTED

  1. Modify the FFmpeg configuration in the admin panel. (Global Video Settings)
  2. We recommend to set up a cronjob to trigger the encoding job manager.
    For more information, please have a look at /video_conversion_cronjob.php.
  3. Edit the video user level settings.
  4. Make sure that the maximum upload filesize is the same in your php.ini file.
  5. Also, make sure that your php.ini is set up correctly to handle big file uploads.
