INSTALLATION INSTRUCTIONS

  1. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server, EXCEPT for install.html, changelog.html, upgrade.html and license.html. Be sure to upload the files to the directory on your server where you have already installed SocialEngine.


  2. Login to your SocialEngine admin panel. On the "View Plugins" page you will see the plugin you have installed. Click the install link.


  3. After the plugin installation, goto "Facebook Connect Settings", and enter API Key and Secret that were obtained from Facebook. For application setup please see our Facebook Application installation instructions Your installation is now complete!

  4. Be sure to delete the file "admin/install_facebookpublisher.php" when you are finished. Leaving this file intact poses a serious security risk!


INTEGRATION INSTRUCTIONS (optional)

  1. Integrating Login Icons on Home Page.

    You will be making 1 easy change to the file templates/home.tpl.
    Open file templates/home.tpl,

    1. FIND (around line 63) 
          </div>
    <div class='portal_spacer'></div>

  {* SHOW HELLO MESSAGE IF USER IS LOGGED IN *}

      Insert on the line before (immediately BEFORE the </div>) 
        {* OPENIDCONNECT 1/1 START *}
    {include file="openidconnect_home_mini.tpl"}
  {* OPENIDCONNECT 1/1 END *}

    That's it.

  2. Integrating Signup Icons on Signup Page.

    You have two options :
    * small signup icons on the top, before the signup form
    * large signup icons on the right side of the signup form


    First option:
    Open file templates/signup.tpl,

    1. FIND (around line 441) 
        
  <form action='signup.php' method='POST'>
  <div class='signup_header'>{lang_print id=681}</div>

      Insert on the line before (immediately BEFORE) 
        {* OPENIDCONNECT 1/1 START *}
    {include file="openidconnect_signup_top.tpl"}
  {* OPENIDCONNECT 1/1 END *}



    Second option:
    Open file templates/signup.tpl,

    1. FIND (around line 441) 
        
  <form action='signup.php' method='POST'>
  <div class='signup_header'>{lang_print id=681}</div>

      Insert on the line before (immediately BEFORE) 
        {* OPENIDCONNECT 1/2 START *}
  {include file="openidconnect_signup_right.tpl"}
  {* OPENIDCONNECT 1/2 END *}

    2. FIND (around bottom of the file) 
        
  <input type='hidden' name='task' value='{$next_task}'>
  <input type='hidden' name='step' value='{$step}'>
  </form>

      Insert on the line after (immediately AFTER) 
        {* OPENIDCONNECT 2/2 START *}
  {include file="openidconnect_signup_right.tpl"}
  {* OPENIDCONNECT 2/2 END *}

    That's it.
