INSTALLATION INSTRUCTIONS

  1. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server, EXCEPT for install.html, license.html. Be sure to upload the files to the directory on your server where you have already installed SocialEngine. Upload all files in ASCII mode.

  2. Set full permissions (CHMOD 777) on all the files in the "/templates" directory.

  3. Login to your SocialEngine admin panel. On the "View Plugins" page you will see the plugin you have installed. Click the install link. Your installation is now complete!

  4. Under the Plugin Menus, click on "Radcodes Settings" and follow instruction on that page.
