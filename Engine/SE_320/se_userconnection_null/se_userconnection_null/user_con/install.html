INSTALLATION INSTRUCTIONS

  1. A fresh copy of the User Connections Plugin will be installed on your server!
    Please take a backup of your site's code and database before you continue.

  2. Please configure your license from the "Files" section of your "My Account" area on SocialEngineAddOns.com.

  3. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server, EXCEPT for install.html and license.html. Be sure to upload the files to the directory on your server where you have already installed SocialEngine. Upload all files in ASCII mode.

  4. Set full permissions (CHMOD 777) on all files in the "/templates" directory.

  5. Login to your SocialEngine admin panel. On the "View Plugins" page you will see the User Connections plugin that you have installed. Click the install link.

  6. Please fill out all the fields mentioned in the installation steps.

  7. Be sure to delete the file "admin/install_userconnection.php" when you are finished. Leaving this file intact poses a serious security risk!

    8. That's it!
