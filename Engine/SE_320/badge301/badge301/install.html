INSTALLATION INSTRUCTIONS

Before you install Badge plugin, you must install these following plugins, which can be downloaded for FREE in your Shop Account Area

  • Radcodes Core Library
  • Radcodes ePayment System

After you install those 2 plugins above, you can proceed with the installation below.

  1. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server, EXCEPT for install.html, license.html. Be sure to upload the files to the directory on your server where you have already installed SocialEngine. Upload all files in ASCII mode.

  2. Set full permissions (CHMOD 777) on all the files in the "/templates" directory.
    Set full permissions (CHMOD 777) on "/uploads_badges" directory.

  3. Login to your SocialEngine admin panel. On the "View Plugins" page you will see the plugin you have installed. Click the install link. Your installation is now complete!
     
  4. Integration for "Special Members" drop-down on TOP MENU.
    Edit "header.tpl" template, put the code below around line 110, (this would make it show up as the last item on TOP-MENU header)
    {include file='badge_topmenu.tpl'}

    so that the end result would look like this
    {/if}
    {include file='badge_topmenu.tpl'}
    <div class='top_menu_link_container_end'>

    Of course you can put it in any position on your TOP-MENU if you like.
    You may also want to make adjustment to these lines as well if your TOP-MENU shows on 2 lines, and break the layout
    {hook_foreach name=menu_main var=menu_main_args complete=menu_main_complete max=XXXX}
    .....
    {hook_foreach name=menu_main var=menu_main_args start=XXXX}

    Just update XXXX to whatever number you prefer, example = 5.

  5. If you would like to have "Recent Badge's Members" entries listed on homepage, do the following.
    1. Edit "home.php", and put in code below
    $badgeassignments = badgeassignment_get_entries(5);
    $smarty->assign_by_ref('badgeassignments', $badgeassignments);

    Where "5" is the number of job posts you want to show.

    2. Edit "home.tpl" template, and put in code below where you like it to show up
    {include file='badgeassignment_recent_entries.tpl'}

    ADVANCED: the "badgeassignment_get_entries" also accept additional parameters for further filtering and control what entries it should return
    badgeassignment_get_entries($limit, $badge_id, $sort, $where)

    where
    - $limit = the number of entries to return
    - $badge_id = the Badge ID in your system. Use 0 (zero) means return all new assignments for all badges, otherwise, only return entries of required Badge Id
    - $sort = default to "badgeassignment_dateapproved DESC", ie approval date in descending order
    - $where = extra condition for query

    NOTE: you can virtually use this on anywhere you like, not just homepage. You can also have more than one recent badge assignment blocks if you use different variables for them.
